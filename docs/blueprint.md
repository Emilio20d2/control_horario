# **App Name**: ShiftMaster

## Core Features:

- Config Definition: Implement a settings page to define company holidays and types of absence
- Employee List: Manage a central store of employees, displaying each employee with a clear listing.
- Employee Detail: Create and view details of each employee, including personal information and contract type.
- Weekly Schedule Display: Visualize the weekly schedule, showing recorded hours.
- Week Navigation: Create a navigable view to move to past or future weeks in the schedule.
- Confirm Button: Allow confirming weekly entries with a 'confirm' button.
- Dashboard UI: Display key performance indicators (KPIs) and alerts on the dashboard.

## Style Guidelines:

- Primary color: Deep Blue (#3498DB) to evoke trust and professionalism.
- Background color: Light gray (#ECF0F1), providing a clean and neutral backdrop.
- Accent color: Orange (#E67E22) to highlight important actions and alerts, creating emphasis.
- Headline font: 'Space Grotesk', sans-serif, suitable for headlines and short amounts of body text; if longer text is anticipated, use this for headlines and 'Inter' for body.
- Body font: 'Inter', grotesque-style sans-serif with a modern, machined, objective, neutral look; suitable for body text.
- Consistent use of flat design icons from 'lucide-react' for a modern, unified look.
- Clean, card-based layout with ample white space to ensure a professional and uncluttered interface.
